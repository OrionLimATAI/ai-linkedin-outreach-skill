# AI LinkedIn Outreach Skill

A public Codex Skill for building responsible AI-assisted B2B LinkedIn outreach workflows.

This repository contains:

- `ai-linkedin-outreach/SKILL.md`: the main Skill instructions
- `ai-linkedin-outreach/agents/openai.yaml`: UI metadata
- `ai-linkedin-outreach/references/source-notes.md`: source attribution and reuse notes

## Source Attribution

This skill is inspired by Oleg's public Claude Cowork outreach resource:

https://www.oleg.ae/claude-cowork-outreach

The Skill content is rewritten as an original workflow and does not copy the source page verbatim.

## Use Cases

- Plan a B2B LinkedIn outreach strategy
- Review a lead list
- Draft personalized connection messages
- Build follow-up sequences
- Manage replies and sales handoffs
- Add safety and quality controls before scaling outreach
